Embedding Inference Chart deployed!

**API Endpoints:**
- `/v1/embeddings` - Generate embeddings
- `/v1/rerank` - Rerank documents (with rerank task)
- `/health` - Health check

**Usage examples:**

# Generate embedding
curl -X POST http://{{ include "embedding-inference.fullname" . }}.{{ .Release.Namespace }}:{{ .Values.service.port }}/v1/embeddings \
  -H "Content-Type: application/json" \
  -d '{
    "input": "Hello world",
    "model": "{{ .Values.model.name }}"
  }'

# Rerank (switch task to "rank")
helm upgrade --install embedding {{ include "embedding-inference.fullname" . }} \
  --set engine.task=rank

**Recommended models:**
- Embedding: BAAI/bge-m3, nomic-ai/nomic-embed-text-v1.5
- Rerank: BAAI/bge-reranker-v2-m3, thena-ai/bge-reranker-v2-m3
